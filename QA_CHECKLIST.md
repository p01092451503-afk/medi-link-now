### 🔴 Critical (즉시 수정 필요)
| 항목 | 설명 | 상태 |

### 🟡 Warning (개선 권장)
| 항목 | 설명 |
|------|------|
| Security Definer View | `hospital_status_with_age` 뷰가 생성자 권한으로 실행 |
| 환자 정보 노출 | `active_ambulance_trips`에서 모든 인증 사용자에게 환자 상태/GPS 노출 |
| GPS 이동 기록 노출 | `location_logs`에서 모든 인증 사용자에게 전체 기사 GPS 기록 노출 |

| 모니터 구독 무방비 | `hospital_monitors`에서 비인증 사용자도 삭제 가능 |
